INSTALLATION GUIDE

__________________________________________________________________________________________________
METHOD 1
**************************************************************************************************
Installing with Visual Studio Professional 2013
1) Download Visual Studio Professional 2013 from https://www.visualstudio.com/en-us/downloads/download-visual-studio-vs.aspx
2) Download QT's Visual Studio 2013 add-in(for QT version 5.0)from "download.qt-project.org/official_releases/vsaddin/qt-vs-addin-1.2.3-opensource.exe"
3) Follow the installation of the QT add-in until complete.
4) Open Visual Studio 2013
5) Open the QT5 tab available at the top of Visual Studio IDE.
6) Press the 'Open QT Project File (.pro)' tab.
7) A File Explorer will open up. Navigate to the Segwei Folder.
8) Select the Segwei.pro file found in the Segwei folder.
9) Build the code.
10) Run the code.
*********************************************************************************************************
__________________________________________________________________________________________________________
METHOD 2
*********************************************************************************************************
Installing with QT Creator 5
1) Download QT Creator version 5 from https://www.qt.io/download-open-source/
2) Install QT Creator.
3) Launch QT Creator
4) Press the 'Open Project' button.
5) A file explorer should open. Navigate to the Segwei folder.
6) Select the Segwei.pro file.
7) Build and run the project as normal.
